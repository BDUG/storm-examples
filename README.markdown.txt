# Big Data User Group Storm example 

This repository contains a variety of examples of using Storm. 

1. Word counting: Different implementations showing the basic storm semantic. 
   This example bases on the storm starter project https://github.com/nathanmarz/storm-starter
2. Delta Stepping: A fundamental delta stepping shortest path implementation. 
   This example bases on the the following concept http://citeseerx.ist.psu.edu/viewdoc/summary?doi=10.1.1.46.2200.
